TLDR;

This website contains all the necessary for reproducibility of this PhD. It is build joining three frameworks:

  • workflowr, the main infrastructure that organizes every analysis step as of Github versions. But instead of showing all commits, it just keep the important ones.
  • targets, a Make-like pipeline toolkit for Statistics and data science in R. It supersedes drake package. This allows to keep track of every step of the data science workflow and helps to understand what needs to be rebuild and what has not changed since last iteration. Every complete analysis shall be registered by one entry in the previous package (workflowr)
  • thesisdown, inspired by the package bookdown, and helps to write the thesis in rmarkdown that is easily compiled in PDF, Doc, EPUB, and allows to change the theme/format for several universities/journals.

Use the big button above to navigate thorough the older commits.

Use the top menu for further information and the thesis itself.
